# homebrew-tap

Homebrew tap for tools by [Christian Del Monte](https://github.com/cdelmonte-zg).

## Available formulas

- **delta-explain** — Make Delta pruning visible: CLI for partition pruning and data skipping in Delta Lake. ([repo](https://github.com/cdelmonte-zg/delta-explain))

## Usage

```bash
brew tap cdelmonte-zg/tap
brew install delta-explain
```

Or in one shot:

```bash
brew install cdelmonte-zg/tap/delta-explain
```

## Updates

Formulas are kept up to date automatically. The release workflow in each upstream tool repo recomputes the SHA256 of the published binaries and pushes a fresh formula here on every tag.
